# LTI provider

Marsha can act as an LTI provider and supports version 1.0 of the standard.

For developers' reference, here are examples of LTI launch requests sent by the most common LMS. If your LMS is missing from this list, please feel free to submit a Pull Request to add it to the list and we will make sure Marsha supports it.
